简介

好久没有写菜谱啦，不过真的很想念大家。今天特意做了一盘蒜蓉小龙虾来馋馋大家。世界杯不远矣，炸鸡、啤酒、小龙虾都准备好了吗？ 说到小龙虾，真的是让大家爱恨交加啊。爱它的美味，却又恨它怎么也不够吃。尤其是每次外卖几乎少的可怜的小龙虾总是让人欲哭无泪。如果，我说如果......能学会自己做小龙虾岂不是一件无比幸福的事。 小龙虾的吃法可谓是五花八门，可是菜菜屋却更爱蒜蓉小龙虾和麻辣小龙虾。扑鼻而来的蒜香，真的是让人忍不住流口水，大蒜独有的香气紧紧勾住味蕾。而麻辣小龙虾除了品尝小龙虾细嫩的肉，满盘的调料也是醉人心头。 蒜蓉小龙虾可谓龙虾吃法中的经典吃法啦，简单不失美味。如果你也喜欢，那就不要错过这道蒜香扑鼻的小龙虾啦

详细步骤

步骤1: 提前准备好食材

步骤2: 用小刷子将小龙虾腹部从头到尾洗刷干净后去掉虾头盖以及虾鳃 叨叨叨：小龙虾腹部很容易掩藏脏物，所以建议清洗干净，为了防止虾的大钳，可以带一个胶皮手套，当然如果技术好就无所谓啦

步骤3: 去掉头盖后，就是去虾线了，虾尾一共有三片，捏住最上面中间的一片，用拇指和食指抓住用力拉出泥肠即可
